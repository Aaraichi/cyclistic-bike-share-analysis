# Data Source Information

## Dataset Used  
This project uses the official **Divvy** (now Cyclistic) public historical trip data.

### Files Included  
- Divvy_Trips_2019_Q1.csv  
- Divvy_Trips_2020_Q1.csv  

These datasets include ride details such as:  
- Ride start/end time  
- Start/end station  
- Rider type (member or casual)  
- Ride duration  
- Location coordinates  

### Provider  
**Motivate International Inc.**  
They make this data publicly available for educational and analysis purposes.

### Privacy  
No personally identifiable information (PII) is included in the data.  
Stations and timestamps are safe and allowed for public use.

### Usage License  
This data is permitted for:  
- Academic projects  
- Portfolio work  
- Analytics practice  

It is *not* permitted for:  
- Commercial resale  
- Re-identification of riders
