# Cyclistic Bike-Share Case Study  
Google Data Analytics Capstone Project  
Author: **Ayoub Aaraichi**  
Date: 2025-11-17  

---

##  Project Overview  
This project analyzes how **annual members** and **casual riders** use Cyclistic bikes differently.  
The goal is to generate insights that help the Cyclistic marketing team convert more casual riders into annual members.

##  Key Tasks Completed  

### 1. **ASK**
Defined the business question:  
“How do annual members and casual riders use Cyclistic bikes differently?”

### 2. **PREPARE**
Loaded and reviewed public Divvy/Cyclistic bike-share data.

### 3. **PROCESS**
Cleaned, merged, and transformed the datasets (Q1 2019 & Q1 2020).

### 4. **ANALYZE**
Calculated:  
- Ride lengths  
- Rides by weekday  
- Rides by hour  
- Rides by month  
- Member vs casual comparisons  

### 5. **SHARE**
Created visualizations in R using **ggplot2**.  
Built a PowerPoint with insights and recommendations.  
Created a reproducible RMarkdown report.

### 6. **ACT**
Recommended marketing actions based on behavior patterns observed.
